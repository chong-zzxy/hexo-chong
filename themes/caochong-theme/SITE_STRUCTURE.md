│  [42 篇文章] [8 个分类] [25 个标签]     │
│  环形进度条 + 数字滚动动画               │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│  技能矩阵 / 标签云                       │
│  - 3D 交互式标签云（鼠标移动响应）       │
│  - 或技能雷达图                          │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│  精选内容（Featured）                    │

**技术实现**:
- Canvas 粒子背景（particles.js 或自定义）

- Intersection Observer 实现滚动触发动画

### 2. 博客（Blog）
**路径**: `/blog/` 或 `/posts/`
**定位**: 文章中心，主要内容展示

#### 2.1 文章列表页
- **路径**: `/blog/` 或 `/blog/page/2/`
- **布局**:
  - 主区域：文章卡片列表（支持封面图）
  - 侧边栏：分类树、标签云、热门文章、搜索框
- **筛选**:
  - 按分类筛选
  - 按标签筛选
  - 按时间排序
- **分页**: 数字分页 + 加载更多（可选）

#### 2.2 文章详情页
- **路径**: `/blog/{category}/{slug}/` 或 `/posts/{slug}/`
- **布局**:
  - 左侧：文章内容（markdown 渲染）
  - 右侧：目录（sticky）、相关文章
- **功能**:
  - 阅读进度条（顶部）
  - 代码高亮 + 复制按钮
  - 图片灯箱
  - 文章信息（日期、分类、标签、字数、阅读时间）
  - 上一篇/下一篇导航

### 5. 搜索（Search）
**路径**: `/search/`
**定位**: 全站内容检索

**功能设计**:
```
┌─────────────────────────────────────────┐
│  [🔍 搜索框 - 大号、醒目]                │
│  实时搜索建议 dropdown                   │
└─────────────────────────────────────────┘

┌──────────────┬──────────────────────────┐
│ 侧边栏筛选   │  搜索结果区域            │
│              │                          │
│ 📁 分类树    │  [结果卡片1]             │
│ 🏷️ 标签云    │  标题 + 摘要 + 高亮关键词│
│ 📅 时间范围  │                          │
│              │  [结果卡片2]             │
│              │                          │
└──────────────┴──────────────────────────┘
```

**技术实现**:
- `hexo-generator-search` 插件生成 `search.json`
- 前端 JavaScript 实现本地搜索
- **预留 AI 智能问答接口**:
  ```javascript
  // 预留接口
  async function aiSearch(query) {
    // TODO: 接入 AI 模型
    const response = await fetch('/api/ai-search', {
      method: 'POST',
      body: JSON.stringify({ query })
    });
    return await response.json();
  }
  ```
- Fuse.js 实现模糊搜索
- 搜索结果高亮

## 🗂️ URL 路径设计

### 方案：扁平化 + 分类前缀（推荐）

```
/                           # 首页
/blog/                      # 博客列表
/blog/yolo-tutorial/        # 文章详情（扁平化）
/blog/page/2/               # 分页

/gallery/                   # 摄影作品集
/gallery/city-night/        # 相册详情

/about/                     # 关于我
/search/                    # 搜索
/archives/                  # 归档
/categories/                # 分类索引
/categories/cv/             # 某分类下文章
/tags/                      # 标签索引
/tags/pytorch/              # 某标签下文章
/guestbook/                 # 留言板
/stats/                     # 统计（可选）
```

**Hexo 配置**:
```yaml
# _config.yml
permalink: blog/:title/
# 或
permalink: blog/:category/:title/  # 如果想包含分类
```

## 🎯 RSS 订阅说明

RSS (Really Simple Syndication) 是一种内容订阅格式，用户可以通过 RSS 阅读器（如 Feedly、Inoreader）订阅你的博客，当你发布新文章时，订阅者会自动收到更新。

**是否需要**:
- 如果希望读者方便订阅 → 需要
- Hexo 可以自动生成 RSS feed (`atom.xml`)
- 建议保留（技术博客标配）
