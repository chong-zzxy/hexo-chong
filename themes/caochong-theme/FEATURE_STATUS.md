## 🎨 设计理念

**核心原则**: 简约而不简单，优雅且实用

1. **一定程度的科技风，但不过于花哨**
   - 粒子背景：科技感，但动画轻柔不抢眼
   - 渐变配色：现代感，但色彩克制
   - 卡片设计：层次分明，但边界柔和

2. **一定程度的简约**
   - 留白充足，信息层次清晰
   - 统一的圆角设计 (12px-20px)
   - 一致的阴影系统
   - 克制的动画效果

3. **蓝色主题贯穿**
   - 主色 #317EFB 用于强调和交互
   - 渐变色增加层次
   - 暗色模式自动适配

## 🚀 快速开始

### 本地预览
```bash
cd /Users/chong/Code/hexo-blog

# 清理缓存
npx hexo clean

# 生成静态文件
npx hexo generate

# 启动服务器
npx hexo server

# 访问 http://localhost:4000
```

### 创建新文章
```bash
npx hexo new post "文章标题"
```

### 创建新相册
1. 创建图片目录: `source/img/gallery/album-name/`
2. 上传照片和封面
3. 创建配置文件: `source/gallery/album-name.md`

详见: `GALLERY_UPLOAD_GUIDE.md`
